Authentication

Request a token using client credentials, then include it on each call.

curl -X POST https://auth.sampleuniversity.edu/oauth2/token \
  -d grant_type=client_credentials \
  -u CLIENT_ID:CLIENT_SECRET

Send the token in the header:

Authorization: Bearer YOUR_TOKEN

Conventions

  • Pagination: page 1..n, page_size default 25 max 100
  • Filtering: query params like ?q=ethics&delivery=eLearning
  • Rate limit: 60 requests / minute
  • Error model (example):
{
  "error": "invalid_request",
  "message": "page_size must be <= 100",
  "request_id": "req_abc123",
  "status": 400
}

Quickstart

Search the Library

curl "https://api.sampleuniversity.edu/v1/courses?q=ethics&page=1&page_size=10" \
  -H "Authorization: Bearer YOUR_TOKEN"

Get a Course

curl "https://api.sampleuniversity.edu/v1/courses/crs_123" \
  -H "Authorization: Bearer YOUR_TOKEN"

Enroll in a Course

curl -X POST "https://api.sampleuniversity.edu/v1/courses/crs_123/enroll" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"attempt":"first"}'

View Your Learning Record

curl "https://api.sampleuniversity.edu/v1/me/record" \
  -H "Authorization: Bearer YOUR_TOKEN"
